â€¢ $f(x-3) = (x-3)^2$ â†’ geser 3 satuan ke kanan

â€¢ $f(x+2) = (x+2)^2$ â†’ geser 2 satuan ke kiri

â€¢ $f(x-2) = 2(x-2) + 1 = 2x - 3$

â€¢ $f(x+3) = 2(x+3) + 1 = 2x + 7$

Translasi Vertikal

Materi

Rumus: $f(x) + k$ dimana $k > 0$ menggeser ke atas, $k < 0$ menggeser ke bawah

Contoh:

Jika $f(x) = x^2$, maka:

â€¢ $f(x) + 4 = x^2 + 4$ â†’ geser 4 satuan ke atas

â€¢ $f(x) - 3 = x^2 - 3$ â†’ geser 3 satuan ke bawah

Untuk fungsi linear:

Jika $f(x) = 3x - 1$, maka:

â€¢ $f(x) + 2 = 3x - 1 + 2 = 3x + 1$

â€¢ $f(x) - 5 = 3x - 1 - 5 = 3x - 6$

â€¢ $-f(x) = -x^2$ â†’ parabola terbuka ke bawah

Refleksi terhadap Sumbu Y

Materi

Rumus: $f(-x)$ â†’ membalik grafik terhadap sumbu y

Contoh:

Jika $f(x) = 2x + 1$, maka:

â€¢ $f(-x) = 2(-x) + 1 = -2x + 1$

Jika $f(x) = 3^x$, maka:

â€¢ $f(-x) = 3^{-x} = \frac{1}{3^x}$

Untuk fungsi kuadrat:

Jika $f(x) = x^2 + 2x + 1$, maka:

â€¢ $f(-x) = (-x)^2 + 2(-x) + 1$

â€¢ $f(-x) = x^2 - 2x + 1$
